# Pregunta: ¿Cómo afrontas un problema técnico que no sabes resolver?

Primero intenta comprender completamente el problema antes de buscar soluciones rápidas.

Divide el problema en partes más pequeñas, revisa documentación oficial y, si es necesario, crea un entorno mínimo reproducible para aislar el error.

Si después de investigar no encuentra solución, pide ayuda aportando contexto claro y explicando lo que ya ha probado.
